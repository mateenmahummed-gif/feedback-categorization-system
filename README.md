 Feedback Categorization System

 📌 Overview
This project uses NLP to classify customer feedback into categories like:
- Product Issue
- Delivery Issue
- Service Issue
- Pricing Issue
- Positive

 ⚙️ Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF
- Logistic Regression

 📊 Outputs
- Accuracy Score
- Confusion Matrix
- Charts & Visualizations

 🚀 How to Run
1. Open the notebook in Google Colab
2. Upload dataset.csv
3. Run all cells

 📈 Results
Model achieves good accuracy and categorizes feedback effectively.
